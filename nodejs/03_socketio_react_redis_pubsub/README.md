# socketio_react_redis_pubsub

This is sample chat web applicaiton with multi websocket server.

## How to install

```
$ docker-compose build
```

## How to debug on local server

```
$ docker-compose up -d
```

Then, acccess to following URL with multiple browsers.

```
http://localhost:3000
http://localhost:3001
http://localhost:3002
```
